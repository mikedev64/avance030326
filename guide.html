1. Introducción y Esquema Web de una Página HTML

Una página web bien estructurada sigue el estándar HTML5. Su esquema básico contiene etiquetas semánticas que le dan sentido a cada parte del documento.

📐 Esquema semántico de una página web

Etiquetas semánticas de HTML5
Etiqueta Descripción Ejemplo de uso
<header> Encabezado de la página o sección. Logo, título, slogan. Logo de la rockola, nombre del sitio
<nav> Menú de navegación con enlaces. Géneros musicales, inicio, contacto
<main> Contenido principal y único de la página. Catálogo de canciones, reproductor
<section> Agrupa contenido relacionado temáticamente. Sección de Rock, Sección de Salsa
<article> Contenido independiente y reutilizable. Tarjeta de una canción específica
<aside> Contenido secundario o complementario. Artistas más populares esta semana
<footer> Pie de página: créditos, contacto, redes. Derechos, contacto de la rockola

💡 Esquema visual de la estructura

┌──────────────────────────────────┐
│           <header>               │  ← Logo + Título "RockolaWeb"
├──────────────────────────────────┤
│             <nav>                │  ← Menú: Inicio | Géneros | Pedir canción
├──────────────┬───────────────────┤
│              │                   │
│   <main>    │    <aside>        │  ← Catálogo | Top canciones
│  <section>  │                   │
│  <article>  │                   │
│              │                   │
├──────────────┴───────────────────┤
│           <footer>               │  ← Contacto | © 2025
└──────────────────────────────────┘
📌 Dato clave: Las etiquetas semánticas NO cambian el aspecto visual (eso lo hace CSS), pero sí mejoran la accesibilidad, el SEO y la mantenibilidad del código.

2. Imágenes en HTML – <img>

La etiqueta <img> es una etiqueta vacía (no tiene etiqueta de cierre). Sus atributos más importantes son:

Atributos de la etiqueta <img>
Atributo Obligatorio Descripción
src ✅ Sí Ruta o URL de la imagen
alt ✅ Sí (accesibilidad) Texto alternativo si la imagen no carga
width ❌ No Ancho en píxeles
height ❌ No Alto en píxeles
title ❌ No Tooltip al pasar el mouse

🎸 Ejemplo: Galería de géneros musicales de la Rockola

Código HTML:

<img src="rock.jpg" alt="Género Rock" width="200" title="Rock Clásico" />
<img src="salsa.jpg" alt="Género Salsa" width="200" title="Salsa y Cumbia" />
<img src="pop.jpg"  alt="Género Pop"  width="200" title="Pop Internacional" />

Resultado en el navegador:

📂 Rutas de imágenes

Tipo de ruta Ejemplo Cuándo usarla
Relativa (misma carpeta) src="rockola.jpg" Imagen en la misma carpeta que el .html
Relativa (subcarpeta) src="img/rockola.jpg" Imagen dentro de una carpeta "img/"
Absoluta (URL) src="https://ejemplo.com/foto.jpg" Imagen alojada en internet

3. Videos en HTML – <video> y YouTube

HTML5 permite insertar videos de dos formas principales:

🎬 Forma 1: Etiqueta <video> (archivo local o URL directa)

Atributos de la etiqueta <video>
Atributo Descripción
src Ruta del archivo de video (.mp4, .webm, .ogg)
controls Muestra los controles de reproducción (play, pausa, volumen)
autoplay Reproduce automáticamente al cargar
loop Repite el video en bucle
muted Inicia sin sonido (necesario para autoplay en navegadores)
width / height Dimensiones del reproductor
poster Imagen de portada mientras el video no se reproduce

Código de ejemplo:

<video src="mi-cancion.mp4" controls width="480" poster="portada.jpg">
  Tu navegador no soporta la etiqueta video.
</video>

▶️ Forma 2: Insertar video de YouTube con <iframe>

Para videos de YouTube, usamos la etiqueta <iframe> con la URL de incrustación que YouTube nos provee: 

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/ID_DEL_VIDEO"
  title="Nombre de la canción"
  allowfullscreen>
</iframe>

🎵 Ejemplo en vivo – Video Musical (YouTube):

🎵 Múltiples fuentes con <source>

<video controls width="480">
  <source src="cancion.mp4"  type="video/mp4" />
  <source src="cancion.webm" type="video/webm" />
  Tu navegador no soporta HTML5 Video.
</video>
💡 Buena práctica: Siempre incluir texto alternativo dentro de <video> para navegadores que no lo soporten.

4. Tablas en HTML

Las tablas organizan datos en filas y columnas. Son perfectas para mostrar catálogos, horarios, estadísticas y listas estructuradas.

🏗️ Estructura básica de una tabla

Etiquetas para construir tablas en HTML
Etiqueta Función Obligatoria
<table> Contenedor principal de la tabla ✅
<caption> Título descriptivo de la tabla ❌ (recomendada)
<thead> Sección de encabezados ❌ (recomendada)
<tbody> Sección del cuerpo de datos ❌ (recomendada)
<tfoot> Sección del pie de la tabla (totales) ❌
<tr> Fila de la tabla (Table Row) ✅
<th> Celda de encabezado (negrita y centrada) ✅ en thead
<td> Celda de dato (Table Data) ✅ en tbody

🔗 Atributos especiales: colspan y rowspan

Atributo Descripción Ejemplo
colspan="N" Fusiona N columnas horizontalmente <td colspan="3">Total</td>
rowspan="N" Fusiona N filas verticalmente <td rowspan="2">Rock</td>

5. Formularios en HTML

Los formularios permiten al usuario ingresar datos que se envían a un servidor. Son fundamentales en cualquier aplicación web.

🏗️ Estructura del formulario

Atributos de la etiqueta <form>
Atributo Valores comunes Descripción
action "procesar.php", "#" URL a donde se envían los datos
method "GET", "POST" Método HTTP de envío
enctype "multipart/form-data" Necesario para subir archivos
name Cualquier texto Identificador del formulario

🔤 Tipos de campos <input>

Tipos de input más usados en HTML5
type="…" Descripción Aplicación en la Rockola
text Campo de texto libre Nombre del cliente
email Valida formato de correo Email para recibir lista de reproducción
password Oculta los caracteres Clave de administrador
number Solo números Cantidad de canciones a pedir
tel Número telefónico Teléfono de contacto del local
date Selector de fecha Fecha del evento musical
time Selector de hora Hora de inicio del show
range Control deslizante numérico Control de volumen
color Selector de color Color de las luces de la rockola
file Subir archivo Subir portada de álbum
checkbox Casilla de verificación (múltiple) Géneros favoritos
radio Botón de opción única Calificar la canción
submit Botón de envío "Pedir canción"
reset Limpia el formulario "Limpiar selección"
hidden Campo oculto (no visible) ID del local / sesión
search Campo de búsqueda Buscar canción por nombre
url Valida formato de URL Link de YouTube de la canción

Elementos HTML de formulario
Etiqueta Uso Cuándo usarla
<input> Campo de entrada (múltiples tipos) La mayoría de los campos
<textarea> Área de texto multilínea Comentarios, descripciones largas

6. Menús de Navegación en HTML

Los menús se construyen con la etiqueta semántica <nav> combinada con listas <ul> / <ol> y enlaces <a>.

📌 Tipos de menús

Tipo Descripción Uso típico
Menú horizontal Ítems en una línea (con CSS se hace horizontal) Barra de navegación superior
Menú vertical Ítems en columna (comportamiento por defecto de UL) Sidebar / menú lateral
Menú desplegable Submenú que aparece al hacer hover Géneros con subgéneros
Breadcrumb Muestra la ruta de navegación Inicio > Géneros > Rock
Menú pie de página Dentro del footer Políticas, contacto, redes

🎵 Ejemplo: Menú principal de la Rockola Digital

Código HTML:

<nav>
  <ul>
    <li><a href="index.html">🏠 Inicio</a></li>
    <li><a href="catalogo.html">🎵 Catálogo</a>
      <ul>
        <li><a href="rock.html">🎸 Rock</a></li>
        <li><a href="salsa.html">💃 Salsa</a></li>
        <li><a href="pop.html">🎤 Pop</a></li>
      </ul>
    </li>
    <li><a href="pedir.html">📋 Pedir canción</a></li>
    <li><a href="ranking.html">🏆 Top 10</a></li>
    <li><a href="contacto.html">📞 Contacto</a></li>
  </ul>
</nav>

Resultado (menú vertical con submenú anidado):

  </ol>
